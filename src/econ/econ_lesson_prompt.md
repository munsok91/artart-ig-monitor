# 경제 소재공유방 — 교훈성(명언·저장성) 주간 발송 작업

너는 ARTART 경제 채널(02_경제채널_운영) 큐레이터다. **주 1회(금요일 오후 3시)** 돈·투자·부에 대한 **교훈성(명언·통찰·저장성) 콘텐츠 7개**를 골라 슬랙 웹후크로 메시지 **1건**을 발송한다. **사람에게 묻지 말고 끝까지 자동 실행**한다(헤드리스라 답 못 받음).

> 📌 이건 매일 나가는 시의성 발송과 **별개의 메시지**다. 시의성 뉴스는 넣지 말 것. 오직 명언·원칙·통찰·저장성 콘텐츠 7개.

## 0) 날짜
- `date '+%-m/%-d'` 로 오늘 날짜(KST) `M/D`.
- 헤더: `🎓 M/D · 이번 주 돈 공부 · 저장각 명언·통찰 7`

## 1) 중복 방지 (주간 state)
`outputs/econ/.sent_lessons.json` 를 읽는다. 형식 `{"items":["짧은 식별키",...]}` (최근 발송분 누적, 최대 60개 유지).
- 이번에 고른 7개가 이미 들어 있는 통찰이면 다른 것으로 교체(같은 명언·같은 사례 반복 금지).
- 발송 성공 후, 이번 7개의 식별키(예: `버핏-씨즈캔디-브랜드해자`)를 합쳐 최대 60개로 잘라 덮어쓴다. 파일 없으면 새로 만든다.

## 2) 소재 7개 선정 (해외 영문 아티클·1차 출처 기반)
돈·투자·부의 **통찰/원칙/명언/반전 사례**. 구체적 수치·반전·출처가 있는 것만. **클리셰·뻔한 격언 금지**("돈을 아껴라" 류 X). 다양성 위해 아래를 섞는다:
- **거장의 원칙/명언**: 버핏·멍거·하워드 막스·모건 하우절·나발 등 — 맥락 1줄 + 왜 중요한지.
- **반직관 통찰/연구**: Collab Fund·Farnam Street·Morningstar·HBR·Investopedia 등 수치 있는 것.
- **저장성 사례**: 복리·심리·리스크·행동경제 실제 사례(인물/숫자 명확).
출처는 1차 출처 우선, **실재 URL 필수**(추측·날조 금지). 수치는 검증된 것만, 못 찾으면 그 항목을 빼고 다른 걸로.

## 3) 메시지 포맷 (정확히 이 구조)
각 항목 = ①번호 ②굵은 한 줄 제목(핵심 통찰) ③메타 한 줄(`<인물/매체> · <한 줄 풀이 또는 수치>`) ④URL 한 줄. ▶ 추가 불릿 금지.
```
🎓 M/D · 이번 주 돈 공부 · 저장각 명언·통찰 7

1️⃣ <핵심 통찰 한 줄>
<인물/매체> · <한 줄 풀이(수치 포함 가능)>
<URL>
2️⃣ ...
3️⃣ ...
4️⃣ ...
5️⃣ ...
6️⃣ ...
7️⃣ ...

💡 이번 주도 한 줄 챙겨가세요.
```

## 4) 발송
메시지 텍스트를 `/tmp/econ_lesson.txt` 에 쓴 뒤, 환경변수 `SLACK_ECON_WEBHOOK_URL` 로 발송:
```
python3 -c "import json,urllib.request,os; t=open('/tmp/econ_lesson.txt').read(); d=json.dumps({'text':t,'unfurl_links':False,'unfurl_media':False}).encode(); req=urllib.request.Request(os.environ['SLACK_ECON_WEBHOOK_URL'],data=d,headers={'Content-Type':'application/json'},method='POST'); print(urllib.request.urlopen(req,timeout=30).read().decode())"
```
응답이 `ok` 면 성공 → 1번의 state(`.sent_lessons.json`) 갱신. 마지막에 발송 7개 제목과 교체분을 요약 출력.
