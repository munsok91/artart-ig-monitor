# 에이밍 콘텐츠 플레이북 — 카드뉴스 제작 규칙

> 채널: **@aim___ing** — "대한민국 1등 경제 소식 전달 채널" (B2B 클라이언트: AIM 투자앱)
> 타겟: 한국 20대 중반 ~ 30대 중반 (경제/시장에 관심 있는 사회초년생)
> 소재: 주식, 경제, 글로벌 뉴스, 기업, 명언, 투자정보, 시의성
> 근거: 노션 "에임(클라이언트)" 가이드 + 베스트 게시물 4개 실측 (2026-07)

---

## 1. 한 회차 = 6~7장 구조

| 슬라이드 | 패턴 | 역할 |
|---------|------|------|
| **1** | A. 커버 | 뉴스사진 풀블리드 + 좌하단 2줄 제목 (둘째 줄 민트) |
| **2~5(6)** | B. 본문 | 사진 + 다크 오버레이 + 중앙정렬 [이모지 소제목 + 3줄 본문] |
| **마지막** | C. CTA | 실계정 공식 CTA 장표 원본 풀블리드 (`assets/aim-cta-official.jpg`, 고정) |

- 본문은 4~5장. 내용이 풍부하면 5장까지.
- 본문 한 장 = 소제목 한 문장 + **정확히 3줄** 분량 글 (실측 기준 한 줄 15~20자).

## 2. 어체 (노션 클라이언트 가이드 — 필수)

- 문장 끝은 **"~습니다."** / **"~요."** 만 사용 (슬라이드 + 캡션 모두)
- 뉴스 브리핑 톤. 짧고 단정하게. 숫자·팩트 중심.
- 소제목 앞에 이모지 1개 (📋 📈 💰 🏦 ⚡ 등 내용과 맞는 것)
- 커버 제목: 2줄. 1줄째 흰색(상황), 2줄째 민트(핵심 사건). 궁금증 유발형도 좋음
  - 실측 예: "저평가 탈출 나선 / 하이닉스 미국 상장 공식화"
  - 실측 예: "억만장자 10명의 / 첫 직업, 뭐였을까?"

## 3. 사진 규칙 (노션 가이드 + 2026-07-11 문석 피드백 — 필수)

- **감도가 후킹이다.** 전광판·건물 전경·회의실 같은 "그냥 있는" 사진 금지. 실계정 베스트는 전부 인물 클로즈업(최태원 얼굴)·아이코닉 컷(젊은 머스크)이었다
- 커버 우선순위: ① 인물 얼굴/손 클로즈업 ② 돈·현물 등 촉각적인 클로즈업 ③ 상징적 현장 컷. 이 셋에서 못 찾으면 소재를 바꿔서라도 찾는다
- **고화질(가로 1200px 이상) 뉴스기사 사진 또는 게티이미지**. 네이버 이미지 검색은 sort=date로 최근 보도사진부터
- 화질 낮으면 나노바나나(AI)에 "화질을 2k로 높여줘"로 업스케일
- 커버 얼굴은 제목(하단 좌측)과 겹치지 않게 배치
- 본문 배경도 내용과 "정확히" 맞는 컷 (글로벌 얘기면 NYSE, 대출 얘기면 창구). 오버레이 55%로 어두워짐을 감안
- 다른 언론사의 그래픽/인포그래픽 이미지는 통째로 쓰지 않는다 (보도사진만)
- 사진은 항상 회차 폴더 `assets/`에 다운로드 후 상대경로 참조 (외부 URL 직접 참조 금지)
- **아트아트 그레인 룩 필수** (2026-07-11 문석 확정): 모든 배경사진에 `scripts/artart_grain.py` 적용 — 아트아트 공식 필름 룩(채도 0.78, 대비 1.10, 블랙리프트, 그레인 커버 27/본문 16, 비네트). 본문 오버레이는 0.45 (그레이딩과 합산 감안)

## 3.5 커버 후킹 공식 (2026-07-11 피드백)

- 팩트 나열형 커버 금지. **1줄째(흰색)에 타겟의 속마음/상황, 2줄째(민트)에 반전 팩트**
  - 약한 예: "1년 새 18조 불었다 / 빚투 38조, 경고등 켜졌다" (그냥 뉴스 제목)
  - 강한 예: "월급 모아선 늦다며 / 빚 38조로 주식 삽니다" (내 얘기 + 숫자 펀치)
- 궁금증 유발형도 좋음: "억만장자 10명의 / 첫 직업, 뭐였을까?" (실계정 베스트)
- 20~30대 사회초년생이 "어 이거 내 얘기다" 해야 넘긴다
- **정확히 2줄로 끝낸다** (2026-07-14 문석 지시 — ep06 커버가 3줄로 터진 뒤 확정).
  각 줄 렌더 폭 한도 936px ≈ 공백 포함 한글 12자. 민트 줄이 길어 줄바꿈되면 3줄이 된다.
  `scripts/check_images.py`가 Pretendard 실측으로 검사해 넘으면 FAIL — 카피를 깎아서 통과시킬 것

## 4. 캡션 규칙 (실측)

- 첫 문단: 핵심 뉴스 요약 2~3문장 ("~습니다" 체)
- 이모지 소제목 단락 2~3개로 상세 전개
- 마지막: AIM 앱 언급 + 해시태그 (경제/재테크/주식 계열 5~10개)

## 5. 금지

- 투자 권유·수익 보장 단언 금지 (정보 전달까지만)
